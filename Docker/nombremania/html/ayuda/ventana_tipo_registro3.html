DNS Avanzada
 


Para usuarios avanzados y aquellos que quieren un control total sobre su dominio. Podrás realizar cuantas redirecciones web y de correo necesites (solo limitado por el tope de consumo de DNS de 200Mb/año: aprox 1 millón de hits). Pero además tendrás acceso al panel de DNS avanzado donde podras, entre otros, crear entradas "A" a las IPs donde alojes los servidores, entradas "MX" a tu servidor de correo y registros "CNAME". También podras usar tecnologías como la de balanceo de servidores o MX encadenados para servidores backup de correo. Todo desde tu panel de control.

INCLUYE:
  • El registro y tasas del dominio por el número de años solicitados.
  • Panel de control y soporte en castellano para la gestión de cambios.
  • Subdominios ilimitados* (XXXX.midominio.com) asociados a página de parking o redirección a otra URL cualquiera.
  • Redirecciones de correo electronico ilimitadas*. El correo enviado a XXXX@tudominio.com se reenviará a la dirección que indiques.
  • Entradas "A" ilimitadas*: (XXXX.midominio.com asociado a la IP xxx.xxx.xxx.xx)
  • Entradas "MX" para gestionar tu/s propio/s servidor de correo.
  • Entradas "CNAME"
  • Balanceo de servidores
  • MX encandenados
